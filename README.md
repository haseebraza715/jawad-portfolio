# Jawad Asif — Portfolio

Personal portfolio site: Supply Chain, Demand Planning, Sustainability & Supplier Compliance. Built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

Connect the repo to [Vercel](https://vercel.com); build and deploy are automatic. Replace `public/Jawad_Asif_CV.pdf` with your actual CV file before or after deployment.

## Project structure

- `app/` — Layout, page, globals, sitemap, robots
- `components/` — Nav, Hero, Section, Experience, Projects, Skills, Education, Certifications, Awards, Contact, Footer, ThemeProvider, JsonLdPerson
- `lib/` — constants (content), utils
